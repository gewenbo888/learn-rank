# Learn · Ranked · 在线学习平台排行榜

35+ online learning platforms scored on **6 weighted criteria** across **7 platform types**.

## Links

- **Live:** [learn.psyverse.fun](https://learn.psyverse.fun)
- **GitHub:** [github.com/gewenbo888/learn-rank](https://github.com/gewenbo888/learn-rank)

## Scoring methodology

Each platform scored **1–10 across 6 criteria**, weighted into a single **Overall Score (0–100)**:

| Criterion | Weight | Why |
|---|---:|---|
| **Content quality** | **22%** | The whole point — bad content wastes the most expensive resource: time. |
| **Career impact** | **22%** | Does completing it actually move the needle on jobs/promotions? |
| Pricing | 15% | Education shouldn't bankrupt you. |
| Certification value | 15% | What employers/peers recognise. |
| Course variety | 13% | Important for finding what you need. |
| User experience | 13% | Friction kills completion. |

## Three sortable columns

- **Overall** — composite weighted score (0-100)
- **Career Impact** — career-impact score (×10)
- **Content Quality** — content-quality score (×10)

Sort by **Career Impact** to see Coursera Pro Certs / AWS / Microsoft Learn / LinkedIn Learning at the top. Sort by **Content Quality** for MIT OCW / Khan Academy / freeCodeCamp / Brilliant. Sort by **Overall** for the balanced view.
